\n\t\t\t\tTeam Fortress 2 Update Released\n\t\t\t\t

December 17, 2015 - TF2 Team

\n\t\t\t\t

\n\t\t\t\tAn update to Team Fortress 2 has been released. The update will be applied automatically when you restart Team Fortress 2. The major changes include:
\r\nThe Tough Break Update has arrived!\r\n

    \r\n
  • Additional information available on the website (http://www.teamfortress.com/toughbreak)\r\n
  • 4 new community maps: Snowycoast, Vanguard, Highpass, and Landfall\r\n
  • 26 new contracts, including contracts which issue loaner weapons \r\n
  • 4 new weapon collections\r\n
  • 2 new community taunts: Badpipes and The Bucking Bronco\r\n
  • 1 new official taunt: Mannrobics\r\n
  • New community cosmetics case with 15 items\r\n
\r\nGeneral\r\n
    \r\n
  • Significant weapon and class balance improvements (see section below for full details)\r\n
  • Smissmas 2015 has begun! Launch Team Fortress 2 to receive your Spirit of Giving badge, Gift-Stuffed Stocking, and Winter Holiday noise maker\r\n
  • Added Smissmas 2015 Mystery Gift. Can be found by opening a Tough Break Weapons Case. Have you been Naughty or Nice?\r\n
  • Added Strange Count Transfer Tool
      \r\n
    • Takes the Strange scores from one item and adds them on to another. The count for the source item is reset to zero. Strange part scores are only transferred and zeroed if a matching Strange part is found. Can only be used between Strange items of the same base type.\r\n
    • Can be Crafted by 2 Strange parts in the crafting menu under Rare Items
    \r\n
  • Updated Case opening item drop algorithm. Going forward players are now less likely to get duplicate items from the same rarity tier on repeated Case openings.\r\n
  • Updated several weapons sounds with consistency fixes from community member Mateusz Grzesik\r\n
  • Updated the scoreboard
      \r\n
    • Re-ordered some of the columns\r\n
    • Added an image to represent player ping values instead of the actual ping values. Added an option in Adv. Options to use the actual ping values.
    \r\n
  • Updated several weapon descriptions to better detail their functionality\r\n
  • \'Gifted by\' Tag can now be removed from items via \'Restore?\' Context action from the Backpack view\r\n
  • Fixed a bug in Mann vs. Machine where Medic bots with unusually long uber effects could have the effect time-out prematurely\r\n
  • Fixed a bug where Unusual Slider was not properly saving\r\n
  • All Strange parts can now be applied to Strange cosmetics items\r\n
  • Strange parts added to the Backpack Context Menu for Strange items\r\n
  • Paints now display both Market and Mann Co. Store prices in the Backpack Context Menu on paintable items\r\n
  • Added case and crate opening animation in Backpack view\r\n
  • Fixed seeing the fire texture on the Captain Space Mann, A Head Full of Hot Air, and The Tomb Readers in DirectX 8\r\n
  • Updated the models/materials for Der Wintermantel and made it paintable\r\n
  • Updated the Caped Crusader to include a team-colored style for the cape\r\n
  • Players can no longer use blast damage to move during the pre-round freeze period
      \r\n
    • Added server convar tf_preround_push_from_damage_enable for servers to utilize the old behavior
    \r\n
  • Fixed the Spy-cicle not melting properly if the Spy is burned while stunned\r\n
  • Added an option in Adv. Options to batch overhead combat text\r\n
\r\nPerformance Improvements\r\n
    \r\n
  • Mac and Linux improvements (Windows Direct3D 9 players have these already)
      \r\n
    • Texture streaming now enabled\r\n
    • Reduced memory consumption by ~500 megs\r\n
    • Faster map loads!
    \r\n
  • Players on Mac can now select \'High\' texture settings\r\n
  • Eliminated many cases of hitching when using painted weapons\r\n
  • Eliminated occasional hitching caused by using mat_picmip 2 (or selecting low quality textures in Video Options -> Advanced)\r\n
  • Massively reduced memory consumption when using painted weapons\r\n
  • Improved particle system performance, boosting overall framerate, especially in particle-heavy situations.\r\n
  • Fixed two most common forms of framerate stuttering / hitching when using popular FPS configs\r\n
  • Added -default launch option to help players triage performance / stability problems.
      \r\n
    • When run with -default, the game will ignore most convar settings and run purely with default options\r\n
    • Upon exit, -default runs do not overwrite the player\'s config.cfg file. If a player wants to save the settings of a default run, they should execute host_writeconfig in the console.
    \r\n
  • Fixed 5 second stall when opening options menu\r\n
  • VR players: You must now pass -vr to launch options to play TF using VR\r\n
  • Fixed multiple cases of FPS stutter related to the sound system\r\n
\r\nMannpower Changes\r\n
    \r\n
  • Flags will become poisonous (mark the carrier for death) 90 seconds after having been stolen\r\n
  • If both team flags are stolen and poisonous, they will return when next dropped\r\n
  • Fixed players being able to grapple during the pre-round freeze period\r\n
  • Reduced the number of flag captures required to win from 10 to 7 on official servers\r\n
  • Regeneration powerup
      \r\n
    • Health regeneration amount changes - Scout, Sniper and Spy increased, Engineer decreased
    \r\n
  • Reflect powerup
      \r\n
    • Added pushback force to players who damage the powerup carrier (except for flame damage)
    \r\n
  • Added overheal effect to players whose max health is increased by a powerup\r\n
\r\nGameplay Changes\r\n
    \r\n
  • All
      \r\n
    • Base weapon switch speed changed from 0.67 seconds to 0.5 seconds\r\n
    • Upon spawning, friendly players will show an outline for 10 seconds\r\n
    • Updated various weapon descriptions to better detail their functionality
    \r\n
  • Multi-Class

    • The Half-Zatoichi
